Specifications:
-------------------------------------------------------------------------------
Complete Programming Problem 8 in the Walls and Mirrors text. (p. 348)

Write a program that will recognize and evaluate prefix expressions.
First design and implement a class of prefix expressions.  This class should 
contain methods to recognize and evaluate prefix expressions.  This chapter
discusses the algorithms you will need to implement these methods.
