Specifications:

Complete Programming Problem 6 in Chapter 7 in the Walls and Mirrors text.  

To complete this problem implement a class Stack that implements the Stack ADT 
StackInterface given on pages 363 - 364 and the class StackReferenceBased given 
on pages 368 - 369 of Walls and Mirrors.    

As the problem describes you should create a class Calculator.  You should then 
also create a class TestDriver that exercises each of the methods in the class 
Calculator.  An explanation should be given as to which methods are being 
tested, the methods should then be tested and the results given showing that 
the methods works correctly.  Be sure to also show and explain examples of 
when the methods throw exceptions.

--- From Book:
6.  Consider simple infix expressions that consist of single-digit operands; 
the operators +, -, *, %, /; and parentheses.  Assume that unary operators 
are illegal and that the expression contains no embedded spaces.

Design and implement a class for an infix calculator.  Use the algorithms given 
in this chapter to convert the infix expression to postfix form and to evaluate 
the resulting postfix expression.  Note that if the methods evaluate and 
getPostfix are called before the convertPostfix method, then the exception 
IllegalStateExceptions should be thrown by these methods.
